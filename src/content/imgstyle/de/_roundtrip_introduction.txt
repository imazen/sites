# Styling of `<img>` Elements and Example Layouts

## Styling of `<img>` Elements

In addition to specifying the source file of an image, the `<img>` element offers some styling options. For example, you can adjust the width and height of the image, align it with the text, or add a border. Here are some examples:

```html
<img src="path/to/image.jpg" alt="Description of image" width="300" height="200">

<img src="path/to/image.jpg" alt="Description of image" style="border: 1px solid black">

<p><img src="path/to/image.jpg" alt="Description of image" style="float: left; margin-right: 1em;">Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed varius dictum dignissim. Sed id scelerisque nibh, at ultrices arcu.</p>
```

## Example Layouts

### Image Gallery

Here is an example gallery layout with multiple images:

```html
<div class="image-gallery">
  <div class="image">
    <img src="path/to/image1.jpg" alt="Description of image">
  </div>
  <div class="image">
    <img src="path/to/image2.jpg" alt="Description of image">
  </div>
  <div class="image">
    <img src="path/to/image3.jpg" alt="Description of image">
  </div>
</div>

<style>
.image-gallery {
  display: flex;
  flex-wrap: wrap;
  justify-content: space-between;
  align-items: center;
}

.image {
  flex-basis: calc(33.33% - 1em);
  margin-bottom: 2em;
  text-align: center;
}

.image img {
  max-width: 100%;
  height: auto;
}
</style>
```

### Image with Text

Here is an example layout where an image is aligned with text:

```html
<div class="image-with-text">
  <img src="path/to/image.jpg" alt="Description of image" style="float: left; margin-right: 1em;">
  <p>Here is some text that is aligned with the image. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed varius dictum dignissim. Sed id scelerisque nibh, at ultrices arcu.</p>
</div>

<style>
.image-with-text img {
  width: 200px;
  height: auto;
}
</style>
```
