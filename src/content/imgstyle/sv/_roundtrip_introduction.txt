# Formatting of the `<img>` element and examples of layouts

## Formatting of the `<img>` Element

Images are often a key part of a webpage's content and design, and understanding how to format the `<img>` element is important for creating visually appealing layouts.

The `<img>` element has several HTML attributes that can be used to control its appearance and behavior. Some of the most commonly used attributes include:

- `src` specifies the URL of the image to be displayed.
- `alt` provides a text description of the image for screen readers and in cases where the image cannot be displayed.
- `width` and `height` control the size of the image. It's generally recommended to define these in CSS rather than using HTML attributes.
- `class` and `id` can be used to apply styles or JavaScript functionality to the element.

Here's an example of using the `<img>` element with some of these attributes:

```html
<img src="example.png" alt="An example image" width="400" height="300" class="bordered" id="example-image">
```

## Examples of layouts

Below are some examples of how the `<img>` element can be used in different layouts:

### Centered image

To center an image within a container, you can use the following CSS:

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

.container img {
  max-width: 100%;
  max-height: 100%;
  margin: auto;
}
```

And the corresponding HTML:

```html
<div class="container">
  <img src="example.png" alt="An example image">
</div>
```

### Image with caption

To add a caption to an image, you can use the following HTML and CSS:

```html
<figure>
  <img src="example.png" alt="An example image">
  <figcaption>An example caption</figcaption>
</figure>
```

```css
figure {
  display: block;
  position: relative;
  margin: 0;
}

figcaption {
  position: absolute;
  bottom: 0;
  left: 0;
  right: 0;
  background-color: rgba(0, 0, 0, 0.5);
  color: #fff;
  padding: 0.5rem;
  font-size: 0.875rem;
  line-height: 1.2;
}
```

### Image grid

To create a grid of images, you can use the following HTML and CSS:

```html
<div class="grid-container">
  <div class="grid-item">
    <img src="example-1.png" alt="Example 1">
  </div>
  <div class="grid-item">
    <img src="example-2.png" alt="Example 2">
  </div>
  <div class="grid-item">
    <img src="example-3.png" alt="Example 3">
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(240px, 1fr));
  grid-gap: 1rem;
}

.grid-item img {
  width: 100%;
  height: auto;
}
```

These are just a few examples of how the `<img>` element can be formatted and used in different layouts. With some creativity and experimentation, you can achieve a variety of visually compelling designs.
