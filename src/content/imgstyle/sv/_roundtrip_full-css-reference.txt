Here is a comprehensive reference of CSS effects and frames that you can use on images, along with examples for each:

1. **Border:** Adds a border around the image.

```css
img {
  border: 2px solid black;
}
```


2. **Box Shadow:** Adds a shadow around the image.

```css
img {
  box-shadow: 0 0 10px rgba(0, 0, 0, 0.5);
}
```


3. **Opacity:** Adjusts the image's transparency.

```css
img {
  opacity: 0.5;
}
```


4. **Grayscale:** Converts the image to grayscale.

```css
img {
  filter: grayscale(100%);
}
```


5. **Sepia:** Applies a sepia tone to the image.

```css
img {
  filter: sepia(100%);
}
```


6. **Blur:** Blurs the image.

```css
img {
  filter: blur(5px);
}
```


7. **Brightness:** Adjusts the image's brightness.

```css
img {
  filter: brightness(50%);
}
```


8. **Contrast:** Adjusts the image's contrast.

```css
img {
  filter: contrast(200%);
}
```


9. **Hue Rotation:** Rotates the hue of the image's color.

```css
img {
  filter: hue-rotate(90deg);
}
```


10. **Invert:** Inverts the colors of the image.

```css
img {
  filter: invert(100%);
}
```


11. **Saturation:** Adjusts the image's saturation.

```css
img {
  filter: saturate(200%);
}
```


12. **Sepia:** Applies a sepia tone to the image.

```css
img {
  filter: sepia(100%);
}
```


13. **Drop Shadow:** Adds a drop shadow to the image.

```css
img {
  filter: drop-shadow(2px 2px 5px rgba(0, 0, 0, 0.5));
}
```


14. **Transition:** Adds a transition effect to the image.

```css
img {
  transition: transform 0.3s ease;
}
img:hover {
  transform: scale(1.1);
}
```


15. **Animation:** Adds an animation effect to the image.

```css
@keyframes slidein {
  from {
    transform: translateX(100%);
  }
  to {
    transform: translateX(0);
  }
}
img {
  animation: slidein 1s ease;
}
```



These are just a few examples of the many CSS effects and frames you can use on images. Experiment with different styles to create unique and eye-catching designs!
