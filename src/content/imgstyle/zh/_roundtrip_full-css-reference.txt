This is a complete CSS effects and borders reference with examples that you can apply to images:
1. **Border:** Add a border around the image.

```css
img {
  border: 2px solid black;
}
```


2. **Box Shadow:** Add a shadow around the image.

```css
img {
  box-shadow: 0 0 10px rgba(0, 0, 0, 0.5);
}
```


3. **Opacity:** Adjust the opacity of the image.

```css
img {
  opacity: 0.5;
}
```


4. **Grayscale:** Convert the image to a grayscale.

```css
img {
  filter: grayscale(100%);
}
```


5. **Sepia:** Apply a sepia effect to the image.

```css
img {
  filter: sepia(100%);
}
```


6. **Blur:** Blur the image.

```css
img {
  filter: blur(5px);
}
```


7. **Brightness:** Adjust the brightness of the image.

```css
img {
  filter: brightness(50%);
}
```


8. **Contrast:** Adjust the contrast of the image.

```css
img {
  filter: contrast(200%);
}
```


9. **Hue Rotation:** Rotate the hue of the image.

```css
img {
  filter: hue-rotate(90deg);
}
```


10. **Invert:** Invert the colors of the image.

```css
img {
  filter: invert(100%);
}
```


11. **Saturation:** Adjust the saturation of the image.

```css
img {
  filter: saturate(200%);
}
```


12. **Sepia:** Apply a sepia effect to the image.

```css
img {
  filter: sepia(100%);
}
```


13. **Drop Shadow:** Add a shadow effect around the image.

```css
img {
  filter: drop-shadow(2px 2px 5px rgba(0, 0, 0, 0.5));
}
```


14. **Transition:** Add a transition effect to the image.

```css
img {
  transition: transform 0.3s ease;
}
img:hover {
  transform: scale(1.1);
}
```


15. **Animation:** Add an animation effect to the image.

```css
@keyframes slidein {
  from {
    transform: translateX(100%);
  }
  to {
    transform: translateX(0);
  }
}
img {
  animation: slidein 1s ease;
}
```



These are just a few examples of many CSS effects and borders that can be applied to images. Try different styles to create unique and eye-catching designs!
