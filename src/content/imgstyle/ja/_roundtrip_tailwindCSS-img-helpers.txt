Tailwind CSS has several helper classes for manipulating `<img>` tags. Here are some of the most commonly used ones:
- `object-contain`: This class scales the image to fit within the container while maintaining its aspect ratio.
- `object-cover`: This class stretches or crops the image to cover the entire container while maintaining its aspect ratio.
- `object-fill`: This class stretches or compresses the image to fill the entire container without maintaining its aspect ratio.
- `object-none`: This class displays the image at its original size.
- `object-center`: This class centers the image horizontally and vertically within the container.
- `object-left`: This class aligns the image to the left of the container.
- `object-right`: This class aligns the image to the right of the container.
- `object-top`: This class aligns the image to the top of the container.
- `object-bottom`: This class aligns the image to the bottom of the container.

These classes can be used together with other Tailwind CSS classes to specify width, height, add borders or shadows, adjust opacity, etc.
