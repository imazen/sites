Below is a comprehensive reference for numerous CSS effects and borders that can be applied to images. Each example is also provided:

1. **Border:** Add a border around an image.

```css
img {
  border: 2px solid black;
}
```


2. **Box Shadow:** Add a shadow around an image.

```css
img {
  box-shadow: 0 0 10px rgba(0, 0, 0, 0.5);
}
```


3. **Opacity:** Adjust the opacity of an image.

```css
img {
  opacity: 0.5;
}
```


4. **Grayscale:** Convert an image to grayscale.

```css
img {
  filter: grayscale(100%);
}
```


5. **Sepia:** Apply a sepia effect to an image.

```css
img {
  filter: sepia(100%);
}
```


6. **Blur:** Blur an image.

```css
img {
  filter: blur(5px);
}
```


7. **Brightness:** Adjust the brightness of an image.

```css
img {
  filter: brightness(50%);
}
```


8. **Contrast:** Adjust the contrast of an image.

```css
img {
  filter: contrast(200%);
}
```


9. **Hue Rotate:** Rotate the hue of an image.

```css
img {
  filter: hue-rotate(90deg);
}
```


10. **Invert:** Invert the colors of an image.

```css
img {
  filter: invert(100%);
}
```


11. **Saturate:** Adjust the saturation of an image.

```css
img {
  filter: saturate(200%);
}
```


12. **Sepia:** Apply a sepia effect to an image.

```css
img {
  filter: sepia(100%);
}
```


13. **Drop-Shadow:** Add a drop shadow effect to an image.

```css
img {
  filter: drop-shadow(2px 2px 5px rgba(0, 0, 0, 0.5));
}
```


14. **Transition:** Add a transition effect to an image.

```css
img {
  transition: transform 0.3s ease;
}
img:hover {
  transform: scale(1.1);
}
```


15. **Animation:** Add an animation effect to an image.

```css
@keyframes slidein {
  from {
    transform: translateX(100%);
  }
  to {
    transform: translateX(0);
  }
}
img {
  animation: slidein 1s ease;
}
```



These are just a few examples of the numerous CSS effects and borders that can be applied to images. Experiment with different styles to create your own eye-catching designs!
