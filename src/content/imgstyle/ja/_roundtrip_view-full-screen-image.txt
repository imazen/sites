To add the "tap to view fullscreen" option to a gallery, you can create an overlay using JavaScript and CSS that displays an image in fullscreen mode when the user taps on it.

Here is a basic approach to achieve this:

1. First, create a container for the gallery and add all the images you want to display.

```html
<div class="gallery">
  <img src="image1.jpg">
  <img src="image2.jpg">
  <img src="image3.jpg">
  <!-- Add more images here -->
</div>
```

2. Next, add CSS to style the gallery and images.

```css
.gallery {
  display: flex;
  flex-wrap: wrap;
}

.gallery img {
  width: 100%;
  height: auto;
  margin: 5px;
}
```

3. Now, use JavaScript to add a click event listener to each image. When the user clicks on an image, JavaScript will create an overlay that displays the image in fullscreen mode.

```javascript
const gallery = document.querySelector('.gallery');
const images = gallery.querySelectorAll('img');

images.forEach((image) => {
  image.addEventListener('click', () => {
    // Create the overlay
    const overlay = document.createElement('div');
    overlay.classList.add('overlay');
    
    // Create the image element
    const fullScreenImage = document.createElement('img');
    fullScreenImage.src = image.src;
    fullScreenImage.classList.add('full-screen-image');
    
    // Add the image to the overlay
    overlay.appendChild(fullScreenImage);
    
    // Add the overlay to the page
    document.body.appendChild(overlay);
  });
});
```

4. Finally, add some CSS to style the overlay and fullscreen image.

```css
.overlay {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background-color: rgba(0, 0, 0, 0.9);
  display: flex;
  justify-content: center;
  align-items: center;
}

.full-screen-image {
  max-width: 100%;
  max-height: 100%;
  object-fit: contain;
}
```

By following these steps, you can create a gallery where users can view images in fullscreen mode by tapping on them.
