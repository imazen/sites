Aquí hay una referencia completa de los efectos y bordes CSS que se pueden aplicar a las imágenes, junto con ejemplos para cada uno:

1. **Borde:** Agrega un borde alrededor de la imagen.

```css
img {
  border: 2px solid black;
}
```

2. **Sombra:** Agrega una sombra alrededor de la imagen.

```css
img {
  box-shadow: 0 0 10px rgba(0, 0, 0, 0.5);
}
```

3. **Opacidad:** Ajusta la transparencia de la imagen.

```css
img {
  opacity: 0.5;
}
```

4. **Escala de grises:** Convierte la imagen a escala de grises.

```css
img {
 filtro: escala-de-grises(100%);
}
```

5. **Sepia:** Aplica un efecto de sepia a la imagen.

```css
img {
 filtro: sepia(100%);
}
```

6. **Desenfoque:** Desenfoca la imagen.

```css
img {
 filtro: desenfoque(5px);
}
```

7. **Brillo:** Ajusta el brillo de la imagen.

```css
img {
  filtro: brillo(50%);
}
```

8. **Contraste:** Ajusta el contraste de la imagen.

```css
img {
 filtro: contraste(200%);
}
```

9. **Rotar tono:** Rota el tono de la imagen.

```css
img {
 filtro: matiz-rotado(90deg);
}
```

10. **Invertir:** Invierte los colores de la imagen.

```css
img {
 filtro: invertir(100%);
}
```

11. **Saturación:** Ajusta la saturación de la imagen.

```css
img {
 filtro: saturar(200%);
}
```

12. **Sepia:** Aplica un efecto de sepia a la imagen.

```css
img {
 filtro: sepia(100%);
}
```

13. **Sombra arroja**: Agrega un efecto de sombra arrojada a la imagen.

```css
img {
 filtro: sombra-arrojada(2px 2px 5px rgba(0, 0, 0, 0.5));
}
```

14. **Transición:** Agrega un efecto de transición a la imagen.

```css
img {
 transición: transformar 0.3s facilidad;
}
img:hover {
 transformar: escala(1.1);
}
```

15. **Animación:** Agrega un efecto de animación a la imagen.

```css
@keyframes deslizar {
 desde {
 transformar: trasladarX(100%);
}
 hacia {
 transformar: trasladarX(0);
}
}
img {
 animación: deslizar 1s facilidad;
}
```

Estos son solo algunos ejemplos de los muchos efectos y bordes CSS que se pueden aplicar a las imágenes. Experimente con diferentes estilos para crear diseños únicos y llamativos!
