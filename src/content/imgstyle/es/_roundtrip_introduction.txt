# `<img>` Element Styles and Layout Examples

## `<img>` Element Styles

The `<img>` element is used to embed images in a web page. Here are some common styles that can be applied to it:

- `border`: sets the border around the image.
- `width`: sets the width of the image.
- `height`: sets the height of the image.
- `margin`: sets the margin around the image.
- `display`: sets how the element is displayed, for example as a block or inline element.
- `float`: sets the horizontal alignment of the element.

## Layout Examples

Here are some layout examples that use the `<img>` element:

### Image Caption

![Image Caption Example](./image-caption.png)

In this layout, the image is accompanied by a caption. The caption is usually placed below the image and in a smaller font size. Add the caption using the `<figcaption>` element.

```html
<figure>
  <img src="image.jpg" alt="Image Description">
  <figcaption>Caption goes here</figcaption>
</figure>
```

### Image Gallery

![Image Gallery Example](./image-gallery.png)

In this layout, multiple images are displayed in a grid-like structure to create an image gallery. Use the `<figure>` element to group each image with a caption.

```html
<div class="image-gallery">
  <figure>
    <img src="image1.jpg" alt="Image Description">
    <figcaption>Caption goes here</figcaption>
  </figure>
  <figure>
    <img src="image2.jpg" alt="Image Description">
    <figcaption>Caption goes here</figcaption>
  </figure>
  <figure>
    <img src="image3.jpg" alt="Image Description">
    <figcaption>Caption goes here</figcaption>
  </figure>
</div>
```

### Image as Link

![Image as Link Example](./image-as-link.png)

In this layout, the image itself is used as a link to another page. Add a link using the `<a>` element.

```html
<a href="destination.html">
  <img src="image.jpg" alt="Image Description">
</a>
```

### Image with Text Overlay

![Image with Text Overlay Example](./image-text-overlay.png)

In this layout, the image is overlaid with text. Use the `<figure>` element to group the image and text together, and use CSS to position the text.

```html
<figure class="image-text-overlay">
  <img src="image.jpg" alt="Image Description">
  <figcaption>Text overlay goes here</figcaption>
</figure>
```
