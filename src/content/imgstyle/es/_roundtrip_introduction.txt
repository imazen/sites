# Styling `<img>` elements and layout examples

## Styling `<img>` elements

`<img>` elements can be styled with CSS using the `img` selector. Some common styling properties include:

- `width` and `height` to specify the dimensions of the image
- `border` to add a border around the image
- `box-shadow` to add a shadow effect to the image
- `margin` and `padding` to adjust the spacing around the image

Here’s an example of CSS code to apply some of these styles to an `<img>` element:

```css
img {
  width: 300px;
  height: 200px;
  border: 1px solid #ddd;
  box-shadow: 2px 2px 5px #ccc;
  margin: 10px;
  padding: 5px;
}
```

This will display the image with a width of 300 pixels, a height of 200 pixels, a gray border, a shadow effect, and some spacing around it.
