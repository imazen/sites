# Formatting of `<img>` elements and layout examples

## Formatting of `<img>` elements

When using the `<img>` element to display images on a webpage, there are a few formatting options available. 

### 1. Setting image dimensions

You can set the dimensions of an image using the `width` and `height` attributes. For example:

```
<img src="example.jpg" alt="Example image" width="500" height="300">
```

### 2. Aligning images

You can use the `align` attribute to specify how the image should be aligned with the surrounding text. The possible values are `left`, `right`, or `middle`. For example:

```
<img src="example.jpg" alt="Example image" align="left">
```

### 3. Adding borders

You can use CSS to add a border around an image using the `border` property. For example:

```
<img src="example.jpg" alt="Example image" style="border: 1px solid black;">
```
