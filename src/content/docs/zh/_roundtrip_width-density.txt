Learn about the `w` and `x` descriptors in `srcset`. Use examples and Markdown.

>The `srcset` is an HTML attribute that allows developers to provide multiple image sources with different resolutions or pixel densities and select the appropriate image based on the display characteristics of the device. The `srcset` attribute uses a combination of a `URL` and either a `w` or `x` descriptor to specify these images.
## Width (`w`) Descriptor

> The `w` descriptor is used to specify the pixel width of an image. We use it when we want to provide the browser with the size of the image so it can choose the appropriate image based on the available screen space.

> The syntax for using the `w` descriptor in `srcset` is as follows:

```html
<img src="small.jpg"
  srcset="small.jpg 500w,
          medium.jpg 1000w,
          large.jpg 2000w"
  alt="Example Image">
```


> In the example above, we provide the browser with three images and their corresponding pixel widths. The browser will choose the image closest to the available screen space.
## Pixel Density (`x`) Descriptor

> The `x` descriptor is used to specify the pixel density of an image, which is the ratio of physical pixels to CSS pixels. We use the `x` descriptor when we want to provide different versions of the same image to offer images with different pixel densities.

> The syntax for using the `x` descriptor in `srcset` is as follows:

```html
<img src="small.jpg"
  srcset="small.jpg 1x,
          medium.jpg 2x,
          large.jpg 3x"
  alt="Example Image">
```


> In the example above, we provide the browser with three images and their corresponding pixel densities. The browser will choose the image with the pixel density closest to the device screen.

> Note that the `w` and `x` descriptors can be used together in the same `srcset` attribute to provide more choices for the browser to choose from.
