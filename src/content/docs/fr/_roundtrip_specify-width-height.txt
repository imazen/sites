Why should you add width and height in HTML? 

> It is recommended to add the width and height attributes in the HTML code of images because it allows the web browser to know the size of the image before it is fully loaded. This allows the browser to reserve the appropriate space on the web page for the image, preventing content from jumping around during image loading.
> 
> In addition, specifying the width and height attributes for images can also help improve overall web page performance. When a browser knows the size of an image in advance, it can more accurately calculate the space needed to display the image and avoid unnecessary rearrangements or redraws of content during loading.
> 
> Overall, using width and height attributes in the HTML code of images can enhance the user experience by accelerating content loading and reducing visual defects.
