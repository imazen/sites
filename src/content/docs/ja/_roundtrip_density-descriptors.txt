Using Srcset with density specification provides an easy and effective way to serve the most suitable images to each user's device, which can improve performance and user experience. This guide explains the basics of using Srcset and density specification and provides a useful markup example.

## What is Srcset with density specification?

Srcset is an HTML attribute designed to specify multiple image sources for a single `<img>` element. Density specification (`x`) is used in combination with Srcset to provide images in different resolutions based on the pixel density of the user’s display.

## When to Use Srcset with Density Specification

Using Srcset with density specification is particularly useful in the following cases:
1. Providing high-resolution images to high DPI displays (e.g., Retina display) while providing low-resolution images to standard displays.
2. Improving page performance by providing the optimal image size for each user's device.

## Implementing Srcset with Density Specification

To use Srcset with density specification, follow these steps:
1. Prepare images with different resolutions.
2. Add the `srcset` attribute to the `<img>` element and include the image sources and corresponding density specifications.
3. Optionally, add the `sizes` attribute to specify the size of the image that should be displayed on the screen.

### Example Markup

The following is an example of how to implement Srcset with density specification in markup.

```html
<img src="example-image-1x.jpg"
     srcset="example-image-1x.jpg 1x, example-image-2x.jpg 2x, example-image-3x.jpg 3x"
     alt="An example image">
```

This markup example includes the following:
- **src**: Default image source to be displayed if the browser does not support Srcset.
- **srcset**: List of image sources and their density specifications (1x, 2x, 3x). The browser selects the optimal image based on the user's display.
- **alt**: Provides a description of the image for accessibility purposes.

With these steps, Srcset with density specification has been successfully implemented to improve the performance and user experience of the website.
