How does a browser determine which image to select from the list of images in the srcset attribute? This is explained step by step using examples for both width-based and density-based selection, depending on the device and viewport.

When using the `srcset` attribute in HTML, the browser uses a set of rules to select the appropriate image from the list of provided image sources. These rules depend on both the display characteristics of the device (resolution, pixel density) and the size of the viewport. With the `srcset` attribute, you can specify different images based on width (using the `w` descriptor) or pixel density (using the `x` descriptor). Let's take a look at examples for each case.

1. Width-based descriptor (`w`):

Assuming the following `srcset` attribute:

```html
<img src="example-small.jpg" srcset="example-small.jpg 400w, example-medium.jpg 800w, example-large.jpg 1600w" alt="Example Image">
```

The browser follows these steps:

a. Determine the device's DPR (device pixel ratio). For example, the standard display has a DPR of 1, while a high-resolution (Retina) display has a DPR of 2 or higher.

b. Calculate the effective width of each image in `srcset` by multiplying the width descriptor by the DPR. For devices with a DPR of 1:

- example-small.jpg: 400 * 1 = 400px
- example-medium.jpg: 800 * 1 = 800px
- example-large.jpg: 1600 * 1 = 1600px

c. Compare the effective width with the width of the viewport. Assuming the viewport width is 420 px, the browser selects the smallest image with an effective width greater than or equal to the viewport width. In this case, `example-medium.jpg` is selected.

2. Density-based descriptor (`x`):

Assuming the following `srcset` attribute:

```html
<img src="example-1x.jpg" srcset="example-1x.jpg 1x, example-2x.jpg 2x, example-3x.jpg 3x" alt="Example Image">
```

The browser follows these steps:

a. Determine the device's DPR (device pixel ratio). For example, the standard display has a DPR of 1, while a high-resolution (Retina) display has a DPR of 2 or higher.

b. Compare the `x` descriptor in `srcset` with the device's DPR. In this case, we have three images with the following descriptors:

- example-1x.jpg: 1x
- example-2x.jpg: 2x
- example-3x.jpg: 3x

c. Select the image with the `x` descriptor closest to the device's DPR. For a device with a DPR of 1, the browser selects `example-1x.jpg`. For a device with a DPR of 2, the browser selects `example-2x.jpg`, and so on.

It's important to note that the `sizes` attribute can be used in combination with the `srcset` attribute to provide more detailed information about how images should be displayed at different viewport widths. This is particularly handy when using width-based descriptors (`w`). Here's an example:

```html
<img src="example-small.jpg" srcset="example-small.jpg 400w, example-medium.jpg 800w, example-large.jpg 1600w" sizes="(max-width: 480px) 100vw, (max-width: 960px) 50vw, ...
```
